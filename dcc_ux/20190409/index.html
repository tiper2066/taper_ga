1. CASE : 에러 메시지 표시 - 직접 표시 / 팝업창 표시 개선안 가이드 보기

로그인 화면과 같이 공간확보가 가능하고 메시지 내용이 적은 경우, 에러메시지를 화면에 직접 표시합니다. 아래 화면에서 로그인 버튼을 클릭해 보십시오.

공간확보가 어렵고 에러메시지의 내용이 많을 경우, 에러가 발생하면 팝업창이 직접 에러 메시지를 표시해 줍니다.
